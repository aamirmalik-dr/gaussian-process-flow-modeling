# Data

## Committed sample

`sample_observations.csv` is a small, license-clean sample generated by a
committed script, `scripts/make_sample.py`. It is synthetic, not carved from any
external product. Each row is one scattered observation of the divergence-free
velocity field:

| column | meaning |
|--------|---------|
| `x`, `y` | observation location in the domain |
| `u_obs`, `v_obs` | measured velocity with additive Gaussian noise (sigma 0.05) |
| `u_true`, `v_true` | noise-free analytic velocity at the same point |

Committing both the noisy measurement and the ground truth lets the offline
quickstart fit on the measurements and still score accuracy and calibration
against the truth, with no field regeneration and no network. Regenerate it with:

```bash
python scripts/make_sample.py --n-obs 300
```

## Everything else is ignored

The rest of this directory is gitignored. This project is fully reproducible from
the synthetic field, so no dataset download is required.
`scripts/download_data.py` documents how a real public ocean-current product (for
example OSCAR or HYCOM sea-surface velocities) could be regridded and loaded into
a `flowgp.field.VelocityField` in place of the synthetic field; the GP
reconstruction and advection code is agnostic to the source of the field.
